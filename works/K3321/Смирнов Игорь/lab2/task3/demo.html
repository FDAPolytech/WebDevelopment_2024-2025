Титульная страница

Чтобы добавить страницу, вставьте её URL в соответствующее текстовое поле и нажмите кнопку Добавить. Для перехода по страницам используйте кнопки Вперёд и Назад по краям демо окна


Чтобы включить автопоказ, введите количество секунд, которое будет отображаться одна страница. Для начала автопоказа нажмите кнопку Старт, а для завершения Стоп
