Чтобы вычислить площадь круга, используйте формулу: πr^2, где r - радиус круга.

Пример кода на JavaScript для вычисления площади круга:

var r = 5; // радиус круга var pi = 3.14; // значение π var area = pi * r * r; // вычисление площади console.log("Площадь круга: " + area);
